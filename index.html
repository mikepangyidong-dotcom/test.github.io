数据来源

示例项目使用公开黄金现货价格 API（如 api.metals.live 等），仅用于学习和展示。若你拥有世界黄金协会 或其他机构的正式数据接口，只需在 app.js 中替换请求地址和解析逻辑即可。

单位与换算

默认展示单位为美元/金衡盎司（USD/oz）。你也可以在脚本中增加换算逻辑，将其转换为人民币/克，并在界面上以 切换按钮的形式呈现。

部署与扩展

页面为纯前端实现，可直接部署在 GitHub Pages。若后续需要更复杂的权限控制或隐藏 API 密钥，建议增加 轻量后端（如 Vercel/Netlify Serverless 函数）作为数据代理层。

本页仅为示例，不构成任何投资建议。数据来源可能存在时延或误差，请以官方渠道为准。
