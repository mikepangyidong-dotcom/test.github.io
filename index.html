这是一个零依赖的静态站点：你可以直接把文件上传到任何静态托管平台（如 Vercel/Netlify/Cloudflare Pages/GitHub Pages），也可以用 Docker 在服务器上运行。

部署 静态 / Docker

现代化 UI

自带基础样式、响应式布局、暗色背景渐变和卡片组件，适合快速改成你的品牌风格。

可扩展

你可以在此基础上升级到 React/Vue/Next.js；也可以加表单、埋点、CDN、甚至后端 API。

可直接上云

支持“直接上传文件”与“Docker + Nginx”两种部署路线，满足从新手到生产的不同需求。

部署方式

  1. 静态托管（最简单）：把本目录下的 index.html、styles.css、 app.js 上传到平台的发布目录即可。
  2. Docker（服务器/容器化）：使用本项目的 Dockerfile 构建镜像，跑 Nginx 来提供静态文件。

详细步骤见 README.md。
